# Contributing to IPLS

Thank you for helping build the multi-century blueprint.

1. All changes must follow the chronological & logical order of operational need.
2. Every page must remain self-contained and ASI-first.
3. Submit Pull Requests against the `main` branch.
4. All contributions are reviewed against the Core Philosophy and Ethical Frameworks 5.1.

See the live Master Spec[](https://ipls.space/spec.php) for full contribution guidelines.
